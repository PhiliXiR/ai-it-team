# Golden Path: Admin Access Request

## Scenario

A contractor needs access to an internal admin tool.

## Flow

1. helpdesk-lead classifies request as access-related
2. iam-lead validates scope and approval state
3. iam-specialist prepares implementation guidance
4. security-director reviews if access is sensitive
5. documentation-lead or change-log-specialist captures the outcome

## What this tests

- intake quality
- approval awareness
- security escalation
- useful output generation
