# Golden Path: Internal App Outage

## Scenario

Multiple users report that an internal application is down.

## Flow

1. incident-triage-specialist gathers initial signals
2. incident-lead frames probable severity
3. systems-lead / network-lead / monitoring-specialist are pulled in as needed
4. ops-director resolves cross-domain ambiguity if required
5. documentation-lead captures incident timeline and final outcome

## What this tests

- incident framing
- coordination quality
- escalation discipline
- documentation quality
