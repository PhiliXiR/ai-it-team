# Runtime Skeleton

This directory is a public-safe placeholder for what a future `ai-it-team` runtime could look like.

## Folders

- `router/` — request classification and routing logic
- `prompts/` — normalized role prompts/specs
- `traces/` — workflow traces and evaluation artifacts
- `logs/` — human-readable run summaries

## Note

This is not a real orchestration runtime yet.
It is a structural step toward one.
